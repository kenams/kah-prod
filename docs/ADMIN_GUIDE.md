# Admin Studio - Guide rapide

## Connexion
- Ouvre `admin.html`.
- Mot de passe par défaut : `kahprod` (modifiable dans `data.js`).

## Modifier le contenu
- Chaque section correspond à une zone du site (Hero, Artistes, Sorties, Réseaux, etc.).
- Les changements sont enregistrés localement dans le navigateur.

## Exporter / Importer
1. Clique sur `Exporter JSON` pour récupérer `kah-prod-data.json`.
2. Pour réimporter, utilise `Importer JSON` et sélectionne ton fichier.

## Publier les modifications
- Le site lit en priorité le contenu du navigateur.
- Pour publier proprement, exporte le JSON et confie-le au dev pour intégration serveur.

## Visuels placeholders
- Les visuels premium par défaut sont dans `assets/placeholders/`.
- Tu peux remplacer chaque URL d'image par tes propres médias (photo, cover, clip).

## Astuces
- Si tu veux repartir de zéro, clique sur `Réinitialiser`.
- Pense à sauvegarder après chaque grosse modification.
